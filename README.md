# @amlplugins/langchain-pinecone

Native `@langchain/pinecone` SDK re-exported under the `@amlplugins` namespace.

> LangChain Pinecone integration — PineconeStore vector store for similarity search and RAG retrieval.

## Install

```bash
npm install @amlplugins/langchain-pinecone
```

## Use

```ts
import { /* named exports from @langchain/pinecone */ } from "@amlplugins/langchain-pinecone";
// or
import sdk from "@amlplugins/langchain-pinecone";
```

This package additionally re-exports `@langchain/core` under the `secondary` namespace.

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
